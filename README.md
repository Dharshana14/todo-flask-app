# To-Do List Web App

This is a simple To-Do List web application built using Flask, HTML, CSS, and SQLite.

## Features
- Add tasks
- Edit tasks
- Delete tasks
- Mark tasks as completed
- Toggle completed status

## Technologies Used
- Python (Flask)
- HTML & CSS
- SQLite

## How to Run
1. Install Flask:
   pip install flask

2. Run the app:
   python app.py

3. Open browser:
   http://127.0.0.1:5000
